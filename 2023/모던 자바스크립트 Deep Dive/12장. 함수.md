# 12장. 함수

## 12.1 함수란?

> 자바스크립트를 정확히 이해하고 사용하기 위한 핵심 개념 일련의 과정을 문(statement)로 구현하고, 코드 블록으로 감싸서 하나의 실행 단위로 정의한 것

```jsx
function add(x, y) {
	return x + y;
}

add(2, 5); // 7
```

### 용어

- 매개변수

  - 함수 내부로 입력을 전달받는 변수 `x, y`

- 인수(입력) `2, 5`

- 반환값(출력) `x+y`

- 함수 이름 `add`

- 함수 정의

  - 함수 선언문을 통해 함수를 정의

  ```jsx
  function add(x, y) {
  	return x + y;
  }
  ```

- 함수 호출

  - 인수를 매개변수를 통해 함수에 전달하며 함수의 실행을 명시적으로 지시

  ```jsx
  var result = add(2, 5);
  console.log(result); //7
  ```



## 12.2 함수를 사용하는 이유

* 코드의 재사용성
* 유지보수의 편의성
* 코드의 신뢰성
* 코드의 가독성
