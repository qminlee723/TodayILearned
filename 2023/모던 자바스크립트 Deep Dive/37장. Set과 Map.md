# 37. Set과 Map

## 37.1 Set

>  `Set` 객체는 중복되지 않는 유일한 값들의 집합이다

배열과 유사하지만 

- 동일한 값을 중복해서 포함할 수 없다
- 요소 순서에 의미가 없다
- 인덱스로 요소에 접근할 수 없다

### 37.1.1 Set 객체의 생성

Set객체는 Set 생성자 함수로 생성되며, 인수를 전달하지 않으면 빈 객체가 생성된다

```javascript
const set = new Set();
console.log(set); // set(0) {}
```

Set 생성자 함수는 이터러블을 인수로 전달받아 Set 객체를 생성한다. 이 때 이터러블의 중복된 값은 Set 객체에 요소로 저장되지 않는다.
