# 5강. 제어 구조

## 01. 제어구조의 이해

### 구조적 프로그래밍 패러다임

- 절차적 프로그래밍 패러다임의 하위 개념
- Edgar Dijkstra가 처음 제안
  - Go To 문은 해로운 것 같다 > 복잡하게 되어 이해하기 어렵다
  - 스파게티 코드
  - 튜링 어워드 땀
- goto문을 사용하지 않고 프로그램을 3가지 구조만으로 구성하는 프로그래밍 패러다임
  - 순차구조(sequence)
  - 선택구조(selection)
  - 반복구조(iteration)
- 프로그램 실행 흐름이 간결하고, 작은 규모로 조직화하기 쉬움



### 순차구조

- 실행의 흐름을 주어지는 명령의 위치적 흐름에 따라 수행하는 구조
  - 명령 라인 위에서 아래로 흐르는 가장 직관적인 구조
  - 일단 첫 단계를 시작하면 마지막 단계까지 수행



### 선택구조

- 특정 영역 내의 명령문에 대한 실행여부를 프로그램 실행 과정 중 결정하는 구조
  - 실행여부는 조건에 따라 결정
- 분기



### 반복구조

- 특정 영역의 명령문을 여러 번 재 실행하는 구조
  - 반복횟수를 조건에 따라 결정



## 02. 순차구조

### 순차 구조

- 실행의 흐름을 주어지는 명령의 위치적 흐름에 따라 수행하는 구조
  - 명령 라인 위에서 아래로 흐르는 가장 직관적인 구조
  - 일단 첫 단계를 시작하면 **마지막 단계까지** 수행



### 삼각형 출력하기(별찍기)

- 다음과 같이 삼각형 모양으로 *을 출력하는 프로그램을 작성하시오

  ![Screenshot 2024-04-03 at 12.25.39 AM](./assets/Screenshot 2024-04-03 at 12.25.39 AM.png)

```python
print("   *")
print("  ***")
print(" *****")
print("*******")
```



### 사용자 입력

- `input()`
- 키보드로부터 입력된 값을 넣어준다
- 사용자로부터 데이터를 입력받는 함수
  - 입력 데이터를 **문자 데이터 타입**으로 반환
  - 함수의 파라미터는 입력 안내문의 목적으로 활용
- Input 함수 예제

```python
rad = input("반지름의 값을 입력하세요: ")
```



### 프로그래밍 에러

- 설계 미숙, 결함 또는 문법 오류로 프로그램이 의도한 대로 결과를 생성하지 못하는 문제

- 구문 오류(syntax error)
  - 문법 체계에 적합하지 않는 명령문 입력 시 발생
- 실행 오류(runtime error)
  - 논리적으로 실행이 불가능한 명령문 작성 시 발생
- 의미 오류(semantic error)
  - 의미적으로 잘못 해석되는 명령문 작성 시 발생



### 데이터 타입 변환

- 데이터 타입을 다른 타입으로 전환해주는 함수
  - 문자열 타입으로 변환 `str` 함수
  - 정수 타입으로 변환: `int` 함수
  - 소수 타입으로 변환: `float` 함수

- 예제

  ```python
  rad = input("반지름을 입력하세요: ")
  rad = int(rad)
  ```

  ```python
  # 함수의 중첩
  rad = int(input("반지름을 입력하세요: "))
  ```



### print 함수의 확장

- 여러 개의 데이터를 단일 함수로 출력 가능

  - 콤마(,)로 파라미터를 구분하여 입력

  - 데이터 사이에 공백(기본값)이 자동으로 추가

  - sep 옵션을 변경하여 공백 변경 가능

    ```python
    print("원뿔의 부피는", vol, "입니다.")
    # 원뿔의 부피는 vol 입니다.
    print("원뿔의", "부피는", vol, "입니다.", sep="")
    # 원뿔의부피는vol입니다.
    print("원뿔의", "부피는", vol, "입니다.", sep="*")
    # 원뿔의*부피는*vol*입니다.
    ```

    

### 원뿔 계산 프로그램 개선

- 사용자로부터 반지름과 높이를 입력받고 계산

```python
# 반지름, 높이 값 할당
rad = int(input("반지름의 값을 입력하세요: "))
hei = int(input("높이 값을 입력하세요: "))
# 부피 출력
# 콤마(,)로 값을 구분하면, 데이터 사이에 공백 하나가 자동적으로 추가
print("부피의 값은", 1/3 * 3.14 * rad ** 2 * hei, "입니다.")
# 겉넓이 출력
# sep=""으로 공백 없앰
print("겉넓이의 값은", 3.14 * rad ** 2 + 3.14 * rad _ hei, "입니다.", sep="")
```
