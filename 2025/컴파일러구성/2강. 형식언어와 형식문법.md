# 2강. 형식언어와 형식문법

## 1. 형식언어의 기초

### 형식언어의 기초

- 형식언어: 프로그래밍 언어
- 문자열: 기호들이 0,1개 이상 나열된 것
- 알파벳: 기호들의 유한 집합



- T*: 공문자열을 포함한 T에 속하는 기호들로 이루어 질 수 있는 모든 문자열의 집합
- T+: 공문자열을 제외한 모든 문자열의 집합

## 3. 문법의 4종류

### chomsky 계층구조

<img src="./assets/Screenshot 2025-11-20 at 9.06.37 PM.png" alt="Screenshot 2025-11-20 at 9.06.37 PM" style="zoom:50%;" />

- type 0
  - 이 세상의 모든 문법 규칙은 0에 포함
- type 1
  - 비위축 문법(타입0에서 제약을 가함)
  - |a| <= |b|
- type 2 (CFG: context free grammar)
  - 타입 1에서 좀 더 제약을 가함
  - 구문 분석을 할 때 많이 사용됨
- type 3 (Regular Grammar)
  - 타입2에서 더 제약을 가함
  - 우선형, 좌선형
  - 정규문법, 어휘분석



### type 1

- 생성 규칙 오른편에 있는 기호들이 왼쪽의 기호들 갯수보다 많으면 type1 만족



### type 2 (CFG)

- 왼쪽은 논 터미널, 오른편은 제약없음
- 단, type1 제약은 만족해야 함



### type 3

- 우선형, 좌선형 2종류 있음
- 단, type1, 2 제약은 만족해야 함
