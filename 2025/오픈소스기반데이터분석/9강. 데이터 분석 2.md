## 1. 통계적 분석 방법론

### 통계적 분석의 기초

- 데이터에서 패턴과 관계 발견하고 이를 수학적으로 모델링하는 접근 방식
  - 기술통계: 데이터의 주요 특성을 요약하고 설명
  - 추론통계: 모집단에 대한 일반화된 결론을 도출
- statsmodels 라이브러리
  - 통계 모델링과 검정을 위한 오픈소스 파이썬 라이브러리
  - 상관관계 분석, 회귀 분석, 시계열 분석, 가설 검정, 분산 분석 등을 지원

### 가설 검정

<img src="./assets/Screenshot 2025-11-21 at 10.50.46 PM.png" alt="Screenshot 2025-11-21 at 10.50.46 PM" style="zoom:50%;" />

- 설정한 가설이 통계적으로 유의미한지를 확인하는 과정
- 가설 검정 과정
  - 가설 설정: 귀무가설(H0)과 대립가설(H1) 정의
  - 통계량 계산: 검정 통계량(t, F, z등)을 계산
  - p값 도출: H0이 참일 때 관찰된 결과가 나올 확률을 계산
  - 결론 도출: p값이 유의수준(0.05)보다 작으면 H0을 기각



### 분산 분석(ANOVA, Analysis of Variance)

- **세 개 이상의 그룹 간 평균 차이를 한 번에 검정하는 통계적 방법**

  - 여러 그룹을 비교할 때 두 그룹씩 t-검정으로 반복 비교하면 제1종 오류가 발생하는 문제를 해결

- 분산 분석 과정

  - 문제 정의

  - F-통계량 계산: 그룹 간 차이와 그룹 내 차이 비교

    <img src="./assets/Screenshot 2025-11-21 at 10.52.24 PM.png" alt="Screenshot 2025-11-21 at 10.52.24 PM" style="zoom:50%;" />

  - p값 도출: F-분포를 이용한 유의성 평가

  - 결론 도출: p < 0.05면 그룹 간 차이 존재 판단

## 3. 딥러닝 기반 분석 방법론

### 딥러닝의 이해

- 딥러닝은 인공 신경망의 여러 층을 통해 데이터를 분석하고 복잡한 패턴을 학습하는 기계학습의 한 분야
  - 개별적인 하나의 퍼셉트론이 얼마나 많은 계산을 유발하는지 볼 수 있음



### 텐서플로우와 케라스

- 텐서플로우 기본 개념
  - 구글이 개발한 오픈소스 기계학습 프레임워크
  - 대규모 신경망을 구축하고 학습할 수 있는 도구를 제공
  - Backpropagation
- 케라스(래핑 라이브러리)
  - 텐서플로우 위에 구축된 고수준 API
  - 복잡한 딥러닝 모델을 간단하고 직관적으로 구성
- 주요 기능
  - 모델 구성 함수: Sequential(), Dense(), Conv2D(), LSTM() 등
  - 모델 학습 및 평가 함수: compile(), fit(), evaluate(), predict()



### CNN을 이용한 이미지 데이터 분석

- CNN은 이미지 처리에 특화된 딥러닝 모델
- 이미지처리에 특화됨



### 생성형 언어 모델(LLM)의 이해

- 트랜스포머 아키텍처
  - 2017년 구글에서 발표한 "Attention is All You Need" 논문에서 소개된 구조
  - 자기 주의(self-attention) 매커니즘을 통해 텍스트의 장거리 의존성을 효과적으로 포착하는 특성
- 학습 방식
  - 제로샷 학습: 특정 작업 예시 없이 맥락만으로 응답 생성
  - 퓨샷 학습: 소수의 예시로 원하는 출력 형식이나 분석 방향 안내
  - 파인튜닝: 특정 도메인이나 작업에 맞게 모델 추가 학습
