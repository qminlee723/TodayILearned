# 13강. 반정형 데이터 분석

## 1. 반정형 데이터의 개념과 특성

### 반정형 데이터

- 완전한 정형 데이터는 아니지만, 일정한 구조와 규칙을 가짐
  - 웹의 기본 구조
  - 인터넷상의 거의 모든 정보가 반정형 데이터
- HTML, XML, JSON

- 웹 데이터 분석, LLM 개발의 훈련 데이터, 데이터 분석/과학 연구 등에 응용



### 웹 스크래핑

- 웹 페이지 다운로드에 끝나지 않고, 다운로드한 웹 페이지에서 원하는 데이터를 정확히 추출하는 과정
  - 크롤링과 다름!
- 웹 스크래핑 관련 라이브러리
  - Selenium
    - 웹 브라우저를 자동화하여 동적 웹 페이지나 로그인이 필요한 페이지에 효과적으로 접근
    - 실제 웹 브라우저를 실행해 클릭, 입력 등 사용자 행동 모방
  - lxml
    - HTML 문서를 파싱하고, 원하는 데이터를 추출
    - HTML의 태그와 속성으로 구성된 구조를 해석하고, XPath를 통해 특정 요소를 선택적으로 추출



### 텍스트 데이터 분석

- 텍스트 데이터 전처리
  - 정규식을 활용한 특수문자 제거
  - 형태소 분석을 통한 품사 분리, 불용어 처리 등의 과정을 통해 텍스트 데이터를 분석 가능한 형태로 정제
- 키워드 분석
  - 단어 빈도 분석, 워드 클라우드 생성, 주요 키워드 추출을 통해 텍스트 데이터에서 의미 있는 정보 추출
- 텍스트 분류 및 시각화
  - 대규모 언어 모델(LLM)을 활용한 텍스트 분류
  - 분류 결과 시각화, 인사이트 도출 과정을 통해 텍스트 데이터의 패턴 발견

### 텍스트 정제

  - 문자를 선택하는 요소, 반복을 나타내는 요소, 문자 범위를 나타내는 요소로 구성

### 형태소 분석

- 형태소
  - 문법적인 의미를 가지는 최소 단위
- 한국어는 교착어.. 하나의 단어가 다양한 어미나 조사와 결합해 여러 의미를 표현할 수 있어 형태소 분석이 특히 중요함
- Kiwi 라이브러리
  - 한국어 형태소 분석을 위한 최신 라이브러리
  - KoNLPy등 기존보다 빠르고 정확
  - 문장의 핵심 의미를 담고 있는 명사 추출을 위한 목적으로 사용



### 게시글 분류

- 문서/문장을 미리 정의된 카테고리로 분류하는 작업
- 대량의 텍스트 데이터 분류를 위해 대규모 언어 모델(LLM)이 도입됨
  - 문맥 이해 및 생성할 수 있는 딥러닝 기반의 AI 모델
  - 텍스트 분류, 요약, 번역, 질의응답 등 자연어 처리 작업 수행
- LLM 구축 환경
  - vLLM 라이브러리를 통해 LLM 모델 로드
  - Hugging Face의 모델 저장소에서 LG에서 개발한 EXAONE 3.5 버전의 2.4B 모델 사용
  - 한국어 처리 능력이 우수, 구글의 T4 GPU에서도 구동
