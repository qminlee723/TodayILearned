# 12강. 정형 데이터 분석

## 1. 정형 데이터의 특성과 분석 가치

### 정형 데이터

- 행과 열로 구성된 명확한 구조
  - 각 열은 특정 속성을 의미하고, 각 행은 개별 레코드를 표현
  - 데이터베이스, 스프레드시트, CSV 파일 등의 형태로 존재
- 분석 결과의 신뢰성과 높은 해석 용이성
  - 구조화된 형태
  - 집계, 필터링, 정렬 등의 기본적인 데이터 처리 작업부터 고급 통계 분석까지 다양한 분석 기법 적용 가능
- 다양한 데이터 통햅해 객관적인 의사결정 지원
  - 고객행동, 운영 효율성, 시장 트렌드 등에 대한 인사이트 제공을 통해 전략적 의사결정의 기반



### 서울시 공공 자전거 따릉이

  - 서울 전역 2,700여개 대여소 운영
  - 누적 이용 건수 1억 9000만건
  - 스마트폰을 통해 간편 이용할 수 있으며, 대여 및 반납시 시스템에 이용 정보 기록

- 서울 열린데이터 광장(data.seoul.go.kr)

  - 대여 일시, 대여 및 반납 대여소 정보, 이용 시간, 이동 거리등 정보

### 시계열 데이터 분석

- 시계열 데이터
  - 시간의 순서에 따라 관측된 데이터의 연속
  - 일정한 시간 간격으로 수집되며, 각 관측값은 시간적 순서를 가지고 있어 이전 관측값들과 의존관계 형성
  - 특정 현상이 시간에 따라 어떻게 변화하는지, 과거 패턴 분석해 미래를 예측하는데 활용
- 시계열 패턴 분석
  - 시간의 흐름에 따른 데이터의 변화를 관찰하고, 주기성이나 트렌드와 같은 시간적 특성 파악



### 데이터 연계(연계분석)

```python
merged_df = pd.merge(daily_usage_df, rain_df, how='left', left_on='기준_날짜', right_on='일시')
```

- `merge` 메소드
  - SQL의 join과 유사한 방식으로 두 DataFrame을 결합하는 기능
  - 공통된 key 값을 기준으로 두 데이터셋의 정보를 하나로 통합하는 과정
  - (예) 자전거 이용량 데이터와 강수량 데이터를 left join 방식으로 결합



### 상관관계 분석의 이해

- **피어슨 상관계수(Pearson Correlation Coefficient)**

  <img src="./assets/Screenshot 2025-11-22 at 11.49.29 PM.png" alt="Screenshot 2025-11-22 at 11.49.29 PM" style="zoom:50%;" />

  - 두 연속형 변수 간의 선형 관계를 측정하는 통계적 지표
  - Pandas의 `corr()` 메소드 사용
  - 피어슨 상관계수 r은 -1에서 1사이의 값으로 표현
    - 1에 가까울수록 강한 양의 상관관계
    - -1에 가까울수록 강한 음의 상관관계
    - 0에 가까울수록 선형적 관계가 약함(독립적)
